Search Info

说明：
----
1.本程序能通过网站地址以IP地址查询网站的基本信息

使用:
-----
```
$ gcc -o get_hostinfo.o -c get_hostinfo.c 
$ sudo ./get_hostinfo <IPaddress> 
```

例如
```
$ sudo ./get_hostinfo 0.0.0.0 
$ sudo ./get_hostinfo www.pornhub.com
```
